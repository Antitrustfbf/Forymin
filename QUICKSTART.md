# 🚀 Быстрый старт Forymin Bot

**Запустите бота за 5 минут!** ⚡

## 📋 Что вам понадобится

- Python 3.8+ на вашем компьютере
- Токен бота от @BotFather в Telegram
- 5 минут свободного времени

## ⚡ Быстрая установка

### 1. Скачайте и установите зависимости
```bash
# Скачайте проект
git clone https://github.com/your-username/forymin.git
cd forymin

# Установите зависимости
pip install -r requirements.txt
```

### 2. Настройте бота
```bash
# Скопируйте файл настроек
cp .env.example .env

# Откройте файл .env и добавьте ваш токен
nano .env  # или любой текстовый редактор
```

**Содержимое .env файла:**
```env
BOT_TOKEN=ваш_токен_здесь
BOT_USERNAME=ваш_username_бота
```

### 3. Запустите бота
```bash
python bot.py
```

**Готово! 🎉** Ваш бот запущен и готов к работе.

## 🔧 Первые шаги

### 1. Добавьте бота в чат
- Найдите вашего бота в Telegram
- Добавьте его в чат как обычного участника
- Назначьте права администратора

### 2. Инициализируйте бота
Отправьте в чат:
```
/start
```

### 3. Проверьте команды
Отправьте:
```
/help
```

## 🎯 Основные команды

| Команда | Описание |
|---------|----------|
| `/start` | Запуск бота |
| `/help` | Справка по командам |
| `/admin` | Панель администратора |
| `/settings` | Настройки чата |
| `/statistics` | Статистика чата |

## 🛡️ Быстрая настройка модерации

### Включите антиспам
```
/anti_spam on
```

### Установите правила
```
/rules_auto set Будьте вежливы и не спамьте
```

### Настройте приветствие
```
/welcome_message set Добро пожаловать в чат!
```

## 🚨 Если что-то пошло не так

### Бот не отвечает?
1. Проверьте, что токен правильный
2. Убедитесь, что бот добавлен в чат
3. Проверьте права администратора

### Ошибки при запуске?
1. Убедитесь, что Python 3.8+ установлен
2. Проверьте, что все зависимости установлены
3. Посмотрите на сообщения об ошибках

### Проблемы с правами?
1. Бот должен быть администратором
2. Нужны права на удаление сообщений
3. Нужны права на блокировку пользователей

## 📱 Тестирование
