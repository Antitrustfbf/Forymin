# Бот для чатов Telegram (RU)

Функции:
- Анти-спам (лимиты сообщений)
- Приветствие новых участников
- Панель настроек для администраторов (`/settings`)
- Русская локализация по умолчанию (поддержка EN)

## Запуск
1. Установите зависимости:
```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```
2. Создайте `.env` на основе `.env.example` и укажите `BOT_TOKEN`.
3. Запустите бота:
```bash
python -m bot.main
```

## Команды
- `/start` — приветствие и помощь
- `/help` — помощь
- `/settings` — панель настроек (для админов чата)
- `/setwelcome <текст>` — задать приветствие (для админов чата)
