# 🤖 Forymin - Умный Telegram бот для модерации чатов

Мощный Telegram бот для управления чатами с расширенными функциями модерации, специально адаптированный для русскоязычных пользователей.

## ✨ Основные возможности

### 🛡️ Модерация и безопасность
- **Автоматическая модерация** сообщений
- **Антиспам защита** с настраиваемыми порогами
- **Автоудаление** нежелательных сообщений
- **Фильтрация** запрещенных слов и ссылок
- **Защита от флуда** и повторяющихся сообщений

### 👥 Управление пользователями
- **Система предупреждений** с автоматическими действиями
- **Временные муты** с настраиваемой длительностью
- **Блокировка** нарушителей
- **Отслеживание активности** пользователей
- **Автоматическое назначение ролей**

### ⚙️ Административные функции
- **Панель администратора** с удобным интерфейсом
- **Настройки чата** через inline кнопки
- **Статистика** активности и модерации
- **Логирование** всех действий
- **Резервное копирование** настроек

### 🌍 Русскоязычная поддержка
- **Полностью на русском языке**
- **Локализованные команды** и сообщения
- **Адаптированные настройки** для русских чатов

## 🚀 Установка и настройка

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd forymin
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
```bash
cp .env.example .env
```

Отредактируйте файл `.env` и добавьте:
- `BOT_TOKEN` - токен вашего бота от @BotFather
- `BOT_USERNAME` - username бота (без @)

### 4. Запуск бота
```bash
python bot.py
```

## 📋 Команды бота

### 👤 Для всех пользователей
- `/start` - Начать работу с ботом
- `/help` - Показать справку
- `/profile` - Ваш профиль
- `/rules` - Правила чата

### 🔧 Для администраторов
- `/admin` - Панель администратора
- `/settings` - Настройки чата
- `/statistics` - Статистика чата
- `/backup` - Резервная копия настроек

### 🛡️ Модерация
- `/warn @username [причина]` - Предупреждение
- `/ban @username [причина]` - Блокировка
- `/unban @username` - Разблокировка
- `/mute @username 1h [причина]` - Временный мут
- `/unmute @username` - Размут

### ⚙️ Настройки модерации
- `/auto_delete` - Автоудаление сообщений
- `/anti_spam` - Настройки антиспама
- `/welcome_message` - Приветственное сообщение
- `/rules_auto` - Автоматические правила

## 🗄️ Структура базы данных

Бот использует SQLite базу данных со следующими таблицами:

- **chats** - Информация о чатах и их настройках
- **users** - Пользователи и их статусы
- **moderation_logs** - Логи действий модерации
- **statistics** - Статистика активности

## ⚙️ Настройка модерации

### Автоматические действия
Бот автоматически выполняет действия на основе серьезности нарушений:

- **Низкая серьезность** - Предупреждение
- **Средняя серьезность** - Предупреждение или мут
- **Высокая серьезность** - Мут или бан

### Настраиваемые параметры
- Порог спама (сообщений в минуту)
- Порог флуда (повторяющихся сообщений)
- Список запрещенных слов
- Белый список разрешенных ссылок
- Длительность автоматических мутов

## 🔧 Конфигурация

Основные настройки находятся в файле `config.py`:

```python
# Настройки по умолчанию для чатов
DEFAULT_CHAT_SETTINGS = {
    'auto_delete': False,
    'auto_delete_delay': 300,  # 5 минут
    'anti_spam': True,
    'max_warnings': 3,
    'welcome_message': 'Добро пожаловать в чат!',
    'rules': 'Будьте вежливы и соблюдайте правила общения.',
    'log_actions': True,
    'auto_moderation': True
}
```
