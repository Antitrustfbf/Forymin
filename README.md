# Termux Forum

Современный веб-форум для сообщества Termux с адаптивным дизайном и полным функционалом.

## 🚀 Особенности

- **Современный дизайн** - Красивый и отзывчивый интерфейс
- **Адаптивность** - Оптимизирован для всех устройств и разрешений
- **Безопасность** - CSRF защита, валидация данных, безопасные сессии
- **Темная тема** - Автоматическое переключение и ручной выбор
- **Поиск** - Полнотекстовый поиск по форуму
- **Категории и теги** - Организация контента
- **Система репутации** - Поощрение активных участников
- **Мобильная оптимизация** - Отличная работа на мобильных устройствах

## 📋 Требования

- PHP 7.4 или выше
- MySQL 5.7 или выше / MariaDB 10.2 или выше
- Веб-сервер (Apache/Nginx)
- Поддержка PDO и PDO_MySQL

## 🛠️ Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/yourusername/termux-forum.git
cd termux-forum
```

### 2. Настройка базы данных

1. Создайте новую базу данных MySQL:
```sql
CREATE DATABASE termux_forum CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

2. Импортируйте структуру базы данных:
```bash
mysql -u root -p termux_forum < database.sql
```

### 3. Настройка конфигурации

Отредактируйте файл `config.php`:

```php
// Конфигурация базы данных
define('DB_HOST', 'localhost');        // Хост базы данных
define('DB_NAME', 'termux_forum');     // Имя базы данных
define('DB_USER', 'your_username');    // Пользователь базы данных
define('DB_PASS', 'your_password');    // Пароль базы данных

// Настройки форума
define('SITE_NAME', 'Termux Forum');   // Название форума
define('SITE_URL', 'http://localhost'); // URL вашего сайта
```

### 4. Настройка веб-сервера

#### Apache (.htaccess уже включен)
Убедитесь, что mod_rewrite включен:
```bash
sudo a2enmod rewrite
sudo systemctl restart apache2
```

#### Nginx
Добавьте в конфигурацию:
```nginx
location / {
    try_files $uri $uri/ /index.php?$query_string;
}
```

### 5. Права доступа

Установите правильные права доступа:
```bash
chmod 755 -R .
chmod 777 -R uploads/  # Если есть папка загрузок
```

## 🚀 Развертывание в Termux

### 1. Установка необходимых пакетов

```bash
pkg update && pkg upgrade
pkg install php mysql apache2
```

### 2. Запуск сервисов

```bash
# Запуск MySQL
mysqld_safe &

# Запуск Apache
httpd
```

### 3. Настройка базы данных

```bash
mysql -u root -p
```

```sql
CREATE DATABASE termux_forum;
CREATE USER 'forum_user'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON termux_forum.* TO 'forum_user'@'localhost';
FLUSH PRIVILEGES;
EXIT;
```

### 4. Импорт структуры

```bash
mysql -u root -p termux_forum < database.sql
```

## 🔧 Настройка и кастомизация

### Изменение цветовой схемы

Отредактируйте CSS переменные в `assets/css/style.css`:

```css
:root {
    --primary-color: #007bff;      /* Основной цвет */
    --secondary-color: #6c757d;    /* Вторичный цвет */
    --success-color: #28a745;      /* Цвет успеха */
    --warning-color: #ffc107;      /* Цвет предупреждения */
    --danger-color: #dc3545;       /* Цвет ошибки */
}
```

### Добавление новых категорий

```sql
INSERT INTO categories (name, description, slug, color, icon, sort_order) 
VALUES ('Название', 'Описание', 'slug', '#цвет', 'fas fa-icon', порядок);
```

### Настройка уведомлений

В `config.php` можно настроить:

- Максимальное количество попыток входа
- Минимальную длину пароля

## 🐛 Устранение неполадок

### Ошибка подключения к базе данных
- Проверьте настройки в `config.php`
- Убедитесь, что MySQL запущен
- Проверьте права доступа пользователя

### Страница не загружается
- Проверьте логи ошибок Apache/Nginx
- Убедитесь, что PHP работает
- Проверьте права доступа к файлам

### Проблемы с сессиями
- Проверьте настройки PHP для сессий
- Убедитесь, что папка для сессий доступна для записи
