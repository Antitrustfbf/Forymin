# 📚 Примеры использования Forymin Bot

Практические примеры настройки и использования бота для различных сценариев модерации.

## 🚀 Быстрый старт

### 1. Первый запуск бота
```bash
# Клонирование репозитория
git clone https://github.com/your-username/forymin.git
cd forymin

# Установка зависимостей
pip install -r requirements.txt

# Настройка переменных окружения
cp .env.example .env
# Отредактируйте .env файл, добавив ваш токен

# Запуск бота
python bot.py
```

### 2. Добавление бота в чат
1. Добавьте бота в ваш чат
2. Назначьте права администратора
3. Отправьте `/start` для инициализации
4. Используйте `/help` для просмотра команд

## 🛡️ Сценарии модерации

### Сценарий 1: Строгая модерация для бизнес-чата

#### Настройки
```bash
# Включение строгой модерации
/anti_spam on
/anti_spam threshold 3
/auto_delete on
/auto_delete set 2m

# Настройка правил
/rules_auto set Добро пожаловать в бизнес-чат! 
Правила: 
- Запрещена реклама без согласования
- Запрещены оскорбления и провокации
- Запрещено распространение спама
- Соблюдайте деловой этикет

# Настройка приветствия
/welcome_message set Добро пожаловать в наш бизнес-чат! 
Пожалуйста, ознакомьтесь с правилами командой /rules
```

#### Результат
- Автоматическое удаление сообщений через 2 минуты
- Строгий контроль спама (максимум 3 сообщения в минуту)
- Четкие правила для участников
- Профессиональное приветствие новых участников

### Сценарий 2: Умеренная модерация для дружеского чата

#### Настройки
```bash
# Включение умеренной модерации
/anti_spam on
/anti_spam threshold 8
/auto_delete off

# Настройка правил
/rules_auto set Добро пожаловать в дружеский чат!
Правила:
- Будьте вежливы друг к другу
- Не спамьте
- Уважайте других участников
- Веселитесь, но не переходите границы

# Настройка приветствия
/welcome_message set Привет! Рады видеть тебя в нашем дружеском чате! 😊
```

#### Результат
- Более мягкий контроль спама (8 сообщений в минуту)
- Сообщения не удаляются автоматически
- Дружелюбная атмосфера
- Простые и понятные правила

### Сценарий 3: Модерация для образовательного чата

#### Настройки
```bash
# Настройка для образовательного контента
/anti_spam on
/anti_spam threshold 5
/auto_delete on
/auto_delete set 10m

# Добавление образовательных слов в белый список
/anti_spam words add образование
/anti_spam words add обучение
/anti_spam words add курс

# Настройка правил
/rules_auto set Добро пожаловать в образовательный чат!
Правила:
- Делитесь полезной информацией
- Задавайте вопросы по теме
- Не отклоняйтесь от темы чата
- Уважайте других участников

# Настройка приветствия
/welcome_message set Добро пожаловать в образовательный чат! 
Здесь вы можете делиться знаниями и учиться новому.
Используйте /rules для ознакомления с правилами.
```

#### Результат
- Оптимальный контроль спама для образовательного контента
- Автоудаление неактуальных сообщений через 10 минут
- Специальные слова для образовательного контента
- Четкая направленность чата

## 👥 Управление пользователями

### Пример 1: Система предупреждений

```bash
# Первое предупреждение
/warn @username Нарушение правил чата

# Второе предупреждение
/warn @username Повторное нарушение

# Третье предупреждение (автоматический мут)
/warn @username Последнее предупреждение
```

### Пример 2: Временные ограничения

```bash
# Мут на 1 час
/mute @username 1h Нарушение правил

# Мут на 30 минут
/mute @username 30m Спам

# Мут на 1 день
/mute @username 1d Серьезное нарушение
```

### Пример 3: Блокировка и разблокировка

```bash
# Блокировка пользователя
/ban @username Нарушение правил чата

# Разблокировка
/unban @username
```

## ⚙️ Настройка автоматической модерации

### Настройка порогов

```bash
# Установка порога спама
/anti_spam threshold 5

# Установка порога флуда (через конфигурацию)
# По умолчанию: 3 повторяющихся сообщения
```

### Настройка запрещенных слов

```bash
# Добавление слов
/anti_spam words add спам
/anti_spam words add реклама
/anti_spam words add scam

# Просмотр списка
/anti_spam words list

# Удаление слов
/anti_spam words remove спам
```

### Настройка автоудаления

```bash
# Включение с задержкой 5 минут
/auto_delete on
/auto_delete set 5m

# Включение с задержкой 1 час
/auto_delete on
/auto_delete set 1h

# Отключение
/auto_delete off
```

## 🔄 Автоматизация

```bash

# Ежедневное резервное копирование
```

## 💡 Советы и рекомендации

### 1. Постепенная настройка
- Начинайте с мягких настроек
- Ужесточайте модерацию по мере необходимости
- Следите за реакцией участников

### 2. Прозрачность
- Объясняйте причины действий модерации
- Публикуйте правила в закрепленном сообщении
- Регулярно напоминайте о правилах

### 3. Гибкость
- Адаптируйте настройки под специфику чата
- Учитывайте размер и активность чата
- Регулярно пересматривайте настройки

### 4. Обучение участников
- Используйте команду /rules для обучения
- Отвечайте на вопросы о модерации
- Создавайте FAQ по правилам
