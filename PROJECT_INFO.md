# 📋 Информация о проекте Forymin Bot

## 🎯 Обзор проекта

**Forymin Bot** - это мощный Telegram бот для модерации чатов, специально разработанный для русскоязычных пользователей. Бот предоставляет комплексные инструменты для управления чатами, автоматической модерации и защиты от спама.

## 🚀 Ключевые особенности

### 🛡️ Модерация и безопасность
- **Автоматическая модерация** сообщений в реальном времени
- **Антиспам защита** с настраиваемыми порогами
- **Фильтрация контента** по запрещенным словам и ссылкам
- **Защита от флуда** и повторяющихся сообщений
- **Автоудаление** нежелательных сообщений

### 👥 Управление пользователями
- **Система предупреждений** с автоматическими действиями
- **Временные муты** с настраиваемой длительностью
- **Блокировка нарушителей** с возможностью разблокировки
- **Отслеживание активности** и статистика пользователей
- **Автоматическое назначение ролей** (member, admin, creator)

### ⚙️ Административные функции
- **Панель администратора** с удобным интерфейсом
- **Настройки чата** через команды и inline кнопки
- **Статистика активности** и модерации
- **Логирование всех действий** для аудита
- **Резервное копирование** настроек

### 🌍 Русскоязычная поддержка
- **Полностью на русском языке** интерфейс и команды
- **Локализованные сообщения** и справка
- **Адаптированные настройки** для русских чатов
- **Понятные команды** для русскоязычных пользователей

### Технологический стек
- **Язык программирования**: Python 3.8+
- **Telegram Bot API**: python-telegram-bot 20.7+
- **База данных**: SQLite3
- **Асинхронность**: asyncio
- **Логирование**: стандартный модуль logging
- **Конфигурация**: python-dotenv

### Модули системы
1. **Основной бот** (`bot.py`) - обработчики команд и сообщений
2. **База данных** (`database.py`) - управление данными чатов и пользователей
3. **Модерация** (`moderation.py`) - алгоритмы обнаружения нарушений
4. **Команды модерации** (`moderation_commands.py`) - административные команды
5. **Утилиты** (`utils.py`) - вспомогательные функции
6. **Конфигурация** (`config.py`) - настройки и тексты

## 🔒 Безопасность и конфиденциальность

### Защита данных
- **Шифрование**: все данные передаются по HTTPS
- **Локальное хранение**: база данных на сервере пользователя
- **Права доступа**: строгая проверка прав администратора
- **Логирование**: аудит всех действий модерации

### Конфиденциальность
- **Личные данные**: не собираются и не передаются
- **Сообщения**: не сохраняются в базе данных
- **Метаданные**: только необходимая информация для модерации
- **GDPR**: соответствует требованиям защиты данных
